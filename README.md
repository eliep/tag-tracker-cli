# Tag Tracker
WIP

## Installation
```bash
$ npm install -g tag-tracker-cli
$ tag --help
```

## Dev
Require nodejs version >= 10

```bash
$ git clone git@github.com:eliep/tag-tracker-cli.git
$ cd tag-tracker-cli
$ npm install .
$ npm run build
$ sudo npm link
```
